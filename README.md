# Detalhes do bootcamp

O bootcamp Impulso JavaScript Evolution é um programa de recrutamento de devs de todo o Brasil e diferentes níveis técnicos para trabalhar na Impulso de forma 100% remota. O Impulso JavaScript Evolution é para qualquer dev que tem como foco evoluir em uma jornada Fullstack e alcançar outro patamar na carreira.
